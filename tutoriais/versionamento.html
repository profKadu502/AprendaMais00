Git e GitHub

Aprenda a versionar seus projetos e enviar para a nuvem.

📌 Conceitos básicos

  • O que é Git?

    Sistema de controle de versão para projetos.

  • O que é GitHub?

    Plataforma para armazenar projetos com Git.

🚀 Criando um repositório

  • Criar repositório no GitHub

    Configuração inicial do projeto online.

  • Conectar projeto local

    Ligando sua máquina ao repositório remoto.

💻 Comandos essenciais

  • git init

    Inicializa o Git no projeto.

  • git add .

    Adiciona arquivos para versionamento.

  • git commit

    Salva uma versão do projeto.

  • git push

    Envia o projeto para o GitHub.

  • Fluxo de trabalho

    git add → commit → push.
