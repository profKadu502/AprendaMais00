Git e GitHub

Versionamento de projetos na prática

⬅ Voltar

📌 O que é Git?

Git é um sistema de controle de versão que permite salvar diferentes versões de um projeto ao longo do tempo.

🌐 O que é GitHub?

O GitHub é uma plataforma online que permite armazenar e compartilhar projetos que utilizam Git.

🚀 Criando um repositório

  1. Acesse o site do GitHub
  2. Clique em "New repository"
  3. Defina um nome para o projeto
  4. Não marque nenhuma opção inicial
  5. Clique em "Create repository"

💻 Comandos principais

1. Inicializar o Git:

git init

2. Adicionar arquivos:

git add .

3. Criar um commit:

git commit -m "mensagem"

4. Enviar para o GitHub:

git push

📌 Primeiro envio


git branch -M main
git push -u origin main

✅ Fluxo do dia a dia
