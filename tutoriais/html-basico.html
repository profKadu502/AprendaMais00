HTML Básico

Aprenda a estrutura de uma página web do zero.

📘 Introdução

HTML (HyperText Markup Language) é a linguagem usada para estruturar páginas web. Com ele, você define títulos, textos, imagens e links.

🧱 Estrutura básica

Veja um exemplo de estrutura de uma página HTML:


<!DOCTYPE html>
<html>
<head>
    <title>Minha página</title>
</head>
<body>
    <h1>Olá mundo</h1>
</body>
</html>

🏷️ Principais tags

  • <h1> → Título
  • <p> → Parágrafo
  • <a> → Link
  • <img> → Imagem

💡 Dica

Sempre organize seu código com identação para facilitar a leitura.
