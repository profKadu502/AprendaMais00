Análise e Projeto de Sistemas

Esta disciplina aborda os fundamentos da Engenharia de Software com foco em análise e projeto de sistemas. O aluno aprenderá a estruturar soluções de software desde o levantamento de requisitos até a modelagem, documentação e implantação de sistemas.

Ementa

Estudo dos conceitos, técnicas e metodologias de Análise e Projeto de Sistemas de Informação, com ênfase em Engenharia de Software, incluindo levantamento de requisitos, modelagem com UML, projeto de software, padrões de desenvolvimento, documentação e implantação de sistemas.

Objetivo Geral

Desenvolver competências relacionadas à especificação, modelagem, projeto, desenvolvimento e gerenciamento de sistemas, capacitando o aluno a estruturar soluções de software de forma organizada e alinhada às boas práticas.

Habilidades Desenvolvidas

  • Compreender o ciclo de vida do software
  • Realizar levantamento e análise de requisitos
  • Modelar sistemas utilizando UML
  • Aplicar princípios de orientação a objetos
  • Utilizar metodologias ágeis e tradicionais
  • Produzir documentação técnica
  • Relacionar padrões de projeto a problemas reais

Conteúdo Programático

  • 1. Introdução – Engenharia de Software e ciclo de vida
  • 2. Requisitos – Funcionais, não funcionais e elicitação
  • 3. UML – Diagramas estruturais e comportamentais
  • 4. Projeto – Arquitetura e padrões de software
  • 5. Implantação – Documentação, testes e manutenção

Metodologia

A disciplina será desenvolvida por meio de aulas expositivas dialogadas, estudos de caso e atividades práticas. Os alunos desenvolverão um projeto integrador aplicando técnicas de análise, modelagem e projeto de software, utilizando ferramentas UML e metodologias ágeis.

Referências

  • PRESSMAN, Roger S. Engenharia de Software. 8ª ed. McGraw-Hill, 2016.
  • SOMMERVILLE, Ian. Engenharia de Software. 10ª ed. São Paulo, 2018.
  • CALAZANS, Angélica Toffano Sedel. Análise e Projeto de Sistemas.
