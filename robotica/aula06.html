O Display de 7 seguimentos

O display de 7 segmentos é um componente eletrônico usado para mostrar números de 0 a 9 e algumas letras simples. Ele é formado por sete pequenos LEDs organizados em forma de “8”, além de um ponto opcional chamado decimal point. Cada LED é chamado de segmento e recebe uma letra de identificação: a, b, c, d, e, f, g.

  • Ele possui 10 pinos (5 de um lado e 5 de outro);
  • Cada pino corresponde à um LED diferente;
  • São 7 seguimentos mais o ponto e os 2 GND's;
  • Cada seguimento funciona como um LED, mas com um formato diferente.

No Arduino, controlar esse display significa acender combinações específicas desses segmentos para formar cada número. Por exemplo, para mostrar o número 1, acendem‑se apenas os segmentos b e c; para o número 8, todos os segmentos são ligados.

No Tinkercad, o funcionamento é muito visual e intuitivo. Você arrasta o display para a área de montagem, conecta cada segmento a um pino digital do Arduino e adiciona resistores para limitar a corrente. Depois, no código, basta definir quais pinos serão ligados ou desligados para formar o número desejado. O simulador permite ver o display acendendo em tempo real, o que facilita o entendimento de como cada segmento contribui para formar os números. É uma forma prática e didática de aprender como o Arduino controla componentes eletrônicos simples usando lógica digital.

Fique atento à ordem dos pinos!

Vamos ver um exemplo de código, o setup continua igual ao que vimos nos projetos anteriores. Aqui configuramos todos os pinos como saída. 

            void setup ();
            {
                pinMode (3, OUTPUT);
                pinMode (4, OUTPUT);
                pinMode (5, OUTPUT);
                pinMode (6, OUTPUT);
                pinMode (7, OUTPUT);
                pinMode (8, OUTPUT);
                pinMode (9, OUTPUT);
            }

Vamos começar mostrando os números em ordem crescente, então no loop vamos mostrar todos pinos ligados com exceção do pino 9 que neste modelo de configuração é o pino do meio. Desta maneira formaremos o número 0 (ZERO) e damos um delay de 1 segundo entre cada número. 

            void loop ();
            {
                digitalWrite (3, HIGH);
                digitalWrite (4, HIGH);
                digitalWrite (5, HIGH);
                digitalWrite (6, HIGH);
                digitalWrite (7, HIGH);
                digitalWrite (8, HIGH);
                digitalWrite (9, LOW);
                delay (1000)/ 
                               
            }
