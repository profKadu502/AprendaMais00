Aula 01

O que é lógica?

Quando se fala em lógica, normalmente pensamos em racionalidade. Muitas vezes também associamos à lógica matemática. A lógica está relacionada à correção do pensamento; também podemos dizer que é a arte de pensar bem.

A lógica está presente no cotidiano de todas as pessoas, por mais que não tomemos conhecimento. Quando realizamos atividades corriqueiras como tomar banho, ir ao cinema, jogar videogame, acessar redes sociais, entre outras, utilizamos a lógica para pensarmos e organizarmos a ordem do que vamos fazer e como vamos fazer.

MENDES, Joice Barbosa; MUNIZ, Rafael da Silva. Lógica de programação com Portugol, p. 5.

No nosso dia a dia, a lógica está presente em praticamente todas as ações. Logo ao acordar, você realiza uma sequência de atividades, muitas delas automaticamente. Por exemplo, você não “pega a escova e espreme na pasta”; o correto é o contrário. Isso mostra que seguimos uma ordem lógica de ações.

Segundo FORBELLONE (2005), sempre que pensamos, a lógica ou a ilógica necessariamente nos acompanham. Quando falamos ou escrevemos, estamos expressando nosso pensamento; portanto, precisamos utilizar a lógica para que essa comunicação seja clara e correta.

Exemplos de lógica no cotidiano

  • Quero pegar a caneta que está na gaveta. A gaveta está fechada; logo, devo abri-la antes de pegar a caneta.
  • Carlos é pai de Caio; logo, Carlos é mais velho que Caio.
  • Preciso ir de bicicleta para a casa da minha tia, que fica do outro lado da cidade. O pneu furou; logo, preciso consertá-lo antes de sair.

Existe lógica na programação?

Sim, e ela é fundamental.

Os computadores são máquinas eletrônicas capazes de executar instruções de forma extremamente rápida. No entanto, diferente do que muitas pessoas imaginam, eles não possuem inteligência própria e não compreendem a linguagem humana de forma natural.

Internamente, o computador funciona utilizando sinais elétricos que podem estar em dois estados: ligado ou desligado. Esses dois estados são representados pelos números 1 (ligado) e 0 (desligado). Esse sistema é chamado de linguagem binária.

Toda informação processada pelo computador — textos, imagens, vídeos e até os programas — é convertida para essa linguagem composta apenas por 0 e 1. Por exemplo, uma letra, como “A”, pode ser representada internamente por uma sequência de bits (dígitos binários), como 01000001.

Quando escrevemos um programa, utilizamos uma linguagem de programação (como Python, Java ou C). Essa linguagem é mais próxima da linguagem humana, mas ainda assim segue regras rigorosas. O código escrito é então traduzido para linguagem binária por programas chamados compiladores ou interpretadores.

Após essa tradução, o processador (CPU) executa as instruções passo a passo. Ele realiza operações simples, como somar, comparar valores e mover dados na memória. A partir da combinação dessas operações básicas, é possível construir programas complexos.

É nesse contexto que a lógica de programação se torna essencial. Antes mesmo de escrever código, é necessário organizar o pensamento de forma clara e estruturada, definindo uma sequência de passos que resolva o problema proposto.

Essa sequência de passos é chamada de algoritmo. Um algoritmo bem definido permite que o computador execute corretamente a tarefa desejada. Por outro lado, se a lógica estiver incorreta, o programa não funcionará como esperado, mesmo que esteja escrito sem erros de sintaxe.

Relação com o cotidiano

A lógica de programação não é algo distante da realidade. Ela segue o mesmo princípio dos exemplos do dia a dia:

  • Primeiro você identifica o problema
  • Depois define os passos para resolvê-lo
  • E, por fim, executa esses passos na ordem correta

Exemplo: Fazer um café

  1. Colocar água no recipiente
  2. Adicionar o pó de café
  3. Aquecer a água
  4. Filtrar

Se a ordem for alterada, o resultado não será o esperado — o mesmo acontece na programação.

Conclusão

A lógica é a base não apenas da programação, mas de qualquer processo que envolva tomada de decisão e resolução de problemas. Desenvolver o raciocínio lógico é o primeiro passo para aprender a programar de forma eficiente.

"Programar não é apenas escrever código, mas saber resolver problemas de forma lógica e eficiente."

Adaptado de FORBELLONE (2005)
