Programação Orientada a Objetos

A disciplina de Programação Orientada a Objetos tem como objetivo introduzir e desenvolver os fundamentos do paradigma orientado a objetos, abordando conceitos essenciais para o desenvolvimento de software moderno.

Ementa

Serão estudados conceitos como classes, objetos, métodos, atributos, encapsulamento, herança e polimorfismo. Além disso, serão abordados classes abstratas, interfaces, coleções de objetos e estruturas de dados, bem como o tratamento de exceções e a persistência de dados em arquivos e bancos de dados.

A disciplina também contempla padrões de projeto e princípios de Clean Code, visando melhorar a qualidade, legibilidade e organização do código. :contentReference[oaicite:1]{index=1}

Objetivo Geral

Capacitar o estudante a compreender e aplicar os fundamentos da Programação Orientada a Objetos no desenvolvimento de softwares, utilizando modelagem UML, boas práticas de programação e integração com bancos de dados.

Habilidades Desenvolvidas

  • Compreender os princípios da orientação a objetos
  • Modelar sistemas utilizando UML
  • Implementar classes, objetos e métodos
  • Aplicar encapsulamento, herança e polimorfismo
  • Utilizar interfaces e classes abstratas
  • Trabalhar com coleções (listas, mapas, conjuntos)
  • Tratar erros e exceções
  • Persistir dados em arquivos e bancos de dados
  • Aplicar padrões de projeto e Clean Code

Organização da Disciplina

  • Unidade I: Introdução à Orientação a Objetos
  • Unidade II: Classes e Objetos
  • Unidade III: Herança, Polimorfismo e Interfaces
  • Unidade IV: Coleções e Exceções
  • Unidade V: Persistência de Dados
  • Unidade VI: Boas práticas e padrões de projeto

A Programação Orientada a Objetos permite organizar sistemas complexos em estruturas mais compreensíveis, reutilizáveis e de fácil manutenção.
