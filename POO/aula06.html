Aula 06 - Instanciamento

Instanciamento

Instanciar é pegar uma classe e gerar um objeto a partir dela.

Não transformamos uma classe em um objeto. Geramos um objeto a partir de uma classe.

Adicionando Atributos e Chamando Métodos

Adicionando atributos

c1 = nova Caneta
c1.cor="Azul”
c1.ponta=0.5
c1.tampada=falso

C1 = NOME DO OBJETO

CANETA = NOME DA CLASSE

COR, PONTA, TAMPADA = ATRIBUTOS

Chamando um método

c1 = nova Caneta
c1.cor="Azul”
c1.ponta=0.5
c1.tampada=falso
c1.rabiscar()

C1 = NOME DO OBJETO

RABISCAR() = MÉTODO

Vamos continuar adicionando atributos no objeto

Após definir cor, ponta e tampada, podemos também chamar um método do objeto.

Atributo

Armazena um valor no objeto. Ex: c1.cor="Azul"

Método

Executa uma ação no objeto. Ex: c1.rabiscar()

Múltiplos Objetos da Mesma Classe

Podemos ter mais de um objeto da mesma classe coexistindo juntos:

Objeto c1 — Caneta Azul

c1 = nova Caneta
c1.cor="Azul”
c1.ponta=0.5
c1.tampada=falso
c1.rabiscar()

C1 = NOME DO OBJETO

CANETA = NOME DA CLASSE

COR, PONTA, TAMPADA = ATRIBUTOS

RABISCAR() = MÉTODO

Objeto c2 — Caneta Vermelha

c2 = nova Caneta
c2.cor="Vermelho”
c2.ponta=1.0
c2.tampada=falso
c2.tampar()

C2 = NOME DO OBJETO

CANETA = NOME DA CLASSE

COR, PONTA, TAMPADA = ATRIBUTOS

TAMPAR() = MÉTODO

Resumo: Classe vs. Objeto

Classe

Define os atributos e métodos comuns que serão compartilhados por um objeto.

Objeto

É uma instância de uma classe.

Classe define atributos e métodos compartilhados.

Instanciar é o processo que cria uma instância da classe.

Objeto é a instância concreta com valores específicos.

Instanciar é pegar uma classe e gerar um objeto a partir dela. Não transformamos uma classe em um objeto — geramos um objeto a partir de uma classe.
